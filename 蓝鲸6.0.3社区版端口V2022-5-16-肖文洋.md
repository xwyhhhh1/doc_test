## 端口

### 一、简述

1.采用环境为社区版





```bash
BK_HTTP_SCHEMA=https
# 访问PaaS平台的域名
BK_PAAS_PUBLIC_URL="https://paas.dianxin.com"
BK_PAAS_PUBLIC_ADDR="paas.dianxin.com:443"
BK_PAAS_PRIVATE_ADDR="paas.service.consul:80"
BK_PAAS_PRIVATE_URL="http://paas.service.consul"
# 访问CMDB的域名
BK_CMDB_PUBLIC_ADDR="cmdb.dianxin.com:443"
BK_CMDB_PUBLIC_URL="https://cmdb.dianxin.com"
# 访问Job平台的域名
BK_JOB_PUBLIC_ADDR="job.dianxin.com:443"
BK_JOB_PUBLIC_URL="https://job.dianxin.com"
BK_JOB_API_PUBLIC_ADDR="jobapi.dianxin.com:443"
BK_JOB_API_PUBLIC_URL="https://jobapi.dianxin.com"
# 访问节点管理下载插件包的URL前缀
BK_NODEMAN_PUBLIC_DOWNLOAD_URL="https://nodeman.dianxin.com:443"
# lesscode 域名
BK_LESSCODE_PUBLIC_ADDR='lesscode.dianxin.com:443'
BK_LESSCODE_PUBLIC_URL='https://lesscode.d.com:443'
```
