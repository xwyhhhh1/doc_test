## 虚拟机快速配置

```shell
vim /etc/sysconfig/network-scrips/ifcfg-xxx
IPADDR=172.17.20.171
NETMASK=255.255.255.0
GATEMASK=172.17.20.1
curl -O http://mirrors.aliyun.com/repo/Centos-7.repo
curl -O http://mirrors.aliyun.com/repo/epel-7.repo
yum -y install bash-completion.noarch 
ystemctl stop firewalld && systemctl disable firewalld#关闭防火墙，取消开机自启
setenforce 0 && sed -i 's/SELINUX=enforcing/SELINUX=disabled/g' /etc/selinux/config#关闭selinux，修改配置文件永久生效
```
